Булевы атрибуты

Атрибуты, которые не требуют значения — их наличие уже включает функцию.

<input type="checkbox" checked>
<button disabled>Неактивно</button>
<input type="text" readonly>
Атрибут Описание
checked Чекбокс или радио отмечены по умолчанию
disabled Элемент отключен (нельзя взаимодействовать)
readonly Поле только для чтения
required Поле обязательно для заполнения
multiple Можно выбрать несколько вариантов
autofocus Автоматический фокус на элементе
hidden Элемент скрыт
